Date Revised: 09/28/2017                                                                                                                                        
Descripiton :                                                                                                                                                   
Create SAS formats that may be used with SAS procedures to label the values of                                                                                  
HCUP diagnosis and procedure groups data elements, including Clinical                                                                                           
Classification Software (CCS) data elements.  For example, the following SAS                                                                                    
statements use the FLGDXCCS format with the Freq Procedure to                                                                                                   
label the values of DXCCS1 in the frequency listing:                                                                                                            
                                                                                                                                                                
    Proc Freq Data = NIS_2015q1q3_CORE ;                                                                                                                        
         Table DXCCS1 ;                                                                                                                                         
         Format DXCCS1  FLGDXCCS. ;                                                                                                                             
    Run ;                                                                                                                                                       
                                                                                                                                                                
The following tables list formats available for selected HCUP data elements.                                                                                    
(Note that some data elements are not always available.)

    Data Element(s)           Format Name(s)                                                                                                                    
    ===============           ==============                                                                                                                    
    DXCCS1-DXCCSnn            FDXCCS. or FDCCSPDX. or FLGDXCCS.                                                                                                 
    E_CCS1-E_CCSn             FDCCSPDX. or FLGDXCCS.                                                                                                            
    PRCCS1-PRCCSnn            FPRCCS. or FPCCSPR. or FLGPRCCS.
